# Ejercicio 1 de Base de Datos

Este es el primer ejercicio de la asignatura de Base de Datos de la universidad.

## Objetivo

Calcular la participación y cardinalidad de las siguientes relaciones y representarlas gráficamente usando la notación de Chen.

## Ejercicios

1. Un libro puede ser escrito por varios autores, y un autor puede escribir varios libros.

2. Una revista es publicada por una editorial, y una editorial puede publicar muchas revistas.

3. Un autobús puede ser conducido por varios conductores, y un conductor puede conducir varios autobuses.

4. Un cine tiene muchas salas, y una sala solo puede estar ubicada en un cine.

5. Muchas películas se proyectan en un cine, y cada proyección está asociada a una sala.

6. Una pizza contiene muchos ingredientes, y un ingrediente puede aparecer en distintas pizzas.

7. Un repartidor entrega muchos pedidos, y un pedido solo puede ser entregado por un repartidor.

8. Un pedido es realizado por un único cliente, y un cliente puede realizar muchos pedidos.

9. Una casa solo puede tener un propietario, pero un propietario puede tener muchas casas.

10. Una propiedad puede tener múltiples contratos de alquiler, pero un contrato de alquiler solo puede pertenecer a una propiedad.

11. Un contrato de alquiler solo puede ser firmado por un inquilino, pero un inquilino puede firmar varios contratos.
